Photogrammetry Digital Characters

1/4

2025 | Agisoft Metashape, Wrap3D, Cinema 4D, Blender, DAZ Studio

I'm building a digital library of authentic Ghanaian characters for animation and interactive media because they don't exist in commercial 3D libraries. Using photogrammetry, I document volunteers from different ethnic groups, creating production-ready 3D assets that animators can use to tell African stories with actual African faces.

I work with volunteers from different regions across Ghana, capturing 120+ photographs per person using controlled lighting. This systematic approach generates the data photogrammetry software needs to build accurate facial geometry.

Raw photogrammetry scans don't work in animation pipelines - they lack rigging and proper topology. I transfer the scanned facial features onto DAZ Studio base models because DAZ allows me to easily modify age, body type, and maintain animation-ready rigs. I then adjust proportions to match each volunteer's physique, creating complete digital doubles.

Four completed characters wearing traditional dresses and contemporary clothing. Each represents a different Ghanaian ethnic group.

This library currently covers several ethnic backgrounds in Ghana. The goal is to create as many characters as possible spanning Ghana's diversity, then expand the methodology across Africa. When animators need authentic African characters, this system provides them.

The Comeback | PowerPoint Animation

2/4

2024 | Microsoft PowerPoint, Freepik Assets

I noticed PowerPoint had animation features nobody uses for anything beyond slide transitions. I wanted to see if I could build a complete film with them.

This animated short follows three F1 cars in a race. The lead car breaks down at the start, restarts multiple times, then overtakes both opponents just before the finish line.

Working in PowerPoint meant solving cinematic problems without proper animation tools. I figured out how to create camera shakes during high-speed sequences, add dynamic rotations for dramatic moments, and layer motion graphics for atmosphere. The constraint forced me to think differently about timing, visual rhythm, and how to communicate speed and tension using limited animation capabilities.

The project started as curiosity about PowerPoint's ignored features and became an exercise in creative problem-solving within constraints.

PBR Auto Mapper

3/4

2025 | Cinema 4D, Python

As a look developer, I map PBR textures in Cinema 4D constantly - basecolor, normal, roughness, metalness, displacement. Doing this manually for every material wastes hours on repetitive technical work instead of creative decisions.

This Cinema 4D plugin automates the texture mapping process. I point it at a folder containing PBR texture sets, and it scans the files, groups them by naming patterns (like "wood_basecolor.png" and "wood_roughness.png"), and presents checkboxes for each material group. After selecting which materials to create, the plugin generates Cinema 4D materials with all texture channels properly mapped and configured for physically-based rendering workflows.

The plugin switches materials from Cinema 4D's default Blinn reflection to GGX distribution, ensuring textures render correctly in modern PBR pipelines. It handles the technical setup automatically so I can focus on lighting and compositional decisions rather than manual channel assignment.

I built this after mapping hundreds of materials for the photogrammetry characters and Akofena project. The plugin saves approximately 3-5 minutes per material, which adds up significantly across production work.

Akofena

4/4

2024 | DAZ Studio, Cascadeur, Cinema 4D, V-Ray, After Effects

I handled look development and compositing for Akofena, a pre-colonial Ashanti narrative animation, while supervising six final-year students through their thesis project. My role focused on establishing the visual approach and executing the technical pipeline for bringing these historical characters to life.

I provided voice acting for one of the characters. This split-screen shows the recorded performance alongside the final animated result, demonstrating how vocal performance translates into character animation.

I performed the physical acting for the protagonist character, using my body mechanics as reference for the animation. This side-by-side comparison shows my live-action performance next to the final animation, which combined auto-physics simulation with manual keyframe animation. Understanding how real human movement translates into digital performance informed the animation approach.

This shows my lighting and compositing process. I developed the lighting setup for each character using multiple light passes in V-Ray, then imported these passes into After Effects to composite the final promotional poster. The lighting needed to feel authentic to pre-colonial Ashanti aesthetics while maintaining production quality.

Supervising this project meant balancing teaching production fundamentals while maintaining visual consistency across six students' contributions through a unified lighting and compositing approach.
